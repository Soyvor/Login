
# Login Page

This application is designed to provide a simple and easy-to-use interface for users to log into a system. It uses the Java Swing framework to create a graphical user interface (GUI) that allows users to enter their login credentials and authenticate themselves.

## Requirements

This application requires Java Development Kit (JDK) version 8 or later to be installed on the system.
## Usage

To use this application, simply run the Login.java file to start the GUI. The user interface will display two input fields for the user to enter their username and password. Once the user enters their credentials, they can click on the "Login" button to authenticate themselves.

If the login credentials are valid, the user will be taken to the main screen of the application. If the credentials are invalid, the user will be prompted to re-enter their login information.
